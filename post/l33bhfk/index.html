单链表的相关要素

单链表

单链表结构体(若无特殊声明，结点为链表存储数据使用，节点为链表的开/尾端)

1typedef struct LNode
2{
3    int data;
4    struct LNode *next;
5}LNode,*LinkList;
  • LNode作为结构体LNode的别名，*LinkList作为结构体指针
  • LNode在使用上通常代指结点，*LinkList使用上作为链表
  • data为存放的数据
  • *next为后继指针，指向后继结点

不带头节点单链表的初始化

1bool InitList(LinkList &L)
2{
3    L=NULL;
4    return true;
5}

带头节点单链表的初始化

给头指针分配一个结点的空间大小，并指向这个结点，作为头节点。

1bool InitList(LinkList &L)
2{
3    L=(LNode *) malloc(sizeof (LNode));
4    if(L==NULL)
5        return false;
6    L->next=NULL;
7    return true;
8}

带头节点与不带头节点的区别

在使用上，不带头结点更为麻烦。不带头节点的头指针指向用于存放数据的结点；带头节点的头指针则指向头节点，该头节点不存放数据，头节点指向的下一个结点存放数据。

带头节点的按位插入数据

传入链表，判断插入结点的位置。为1，则直接在头节点后插入数据；不为1，则通过while循环找到相应结点，并插入数据。

 1//按位插入结点
 2bool InsertList(LinkList &L,int i,int e)
 3{
 4    if(i<0) //判断插入位置是否合法
 5        return false;
 6    LNode *p=L;     //将指针p指向单链表L的头节点
 7    int j=0;
 8    while(p!=NULL && j<i-1) //判断要在哪个结点之后插入结点
 9    {
10        p=p->next;
11        j++;
12    }
13    if(p==NULL) //当前结点为空则不合法
14        return false;
15    LNode *s = (LNode *)malloc(sizeof (LNode)); //为要插入的结点分配空间
16    s->data = e;
17    s->next = p->next;
18    p->next = s;
19    return true;
20}

指定结点的后插操作

传入结点，在该结点后插入数据

 1//指定结点之后插入
 2bool InsertNextNode(LNode *p,int e)
 3{
 4    if(p==NULL)
 5        return false;
 6    LNode *s = (LNode *)malloc(sizeof (LNode));
 7    if(s==NULL)
 8        return false;
 9    s->data = e;
10    s->next = p->next;
11    p->next=s;
12    return true;
13}

指定结点前插入

由于只传入当前结点，无法得知此结点前的地址，若要实现结点前插入，可将当前结点p视为要 “插入的结点”，在当前结点p之后插入一个新的结点s，并将p结点的数据与下一个指向的结点传个结点s。将新插入的数据给结点p，使结点p指向结点s，这样就实现了结点前的插入。

 1//指定结点前插入
 2bool InsertBeforeNode(LNode *p,int e)
 3{
 4    if(p == NULL)
 5        return false;
 6    LNode *s = (LNode *)malloc(sizeof (LNode));
 7    if(s==NULL)
 8        return false;
 9    s->data=p->data;
10    p->data = e;
11    s->next = p->next;
12    p->next = s;
13    return true;
14}

按位删除

 1bool DeleteList(LinkList &L,int i,int &e)
 2{
 3    if(i<1)
 4        return false;
 5    LNode *p=L;
 6    p=L;
 7    int j=0;
 8    while(p!=NULL && j<i-1)
 9    {
10        p=p->next;
11        j++;
12    }
13    if(p==NULL)
14        return false;
15    if(p->next==NULL)
16        return false;
17    LNode *s=p->next;
18    e=s->data;
19    p->next=s->next;
20    free(s);
21    return true;
22}

指定结点删除

由于只传入结点p，无法得知此结点的前一个结点。可将结点p的下一个结点s的值传给结点p，并且使结点p指向s结点所指向的下一个结点，释放结点s。这样就实现了p结点与s结点的"对换"，间接的实现了删除结点 “p”。

 1//删除指定结点
 2bool DeleteNode(LNode *p,int &e)
 3{
 4    if(p == NULL)
 5        return false;
 6    LNode *s=p->next;
 7    e=p->data;
 8    p->data = s->data;
 9    p->next=s->next;
10    free(s);
11}

按位查找

通过传入链表与位序，通过循环找到你所要的结点。

 1LNode * GetElem(LinkList L,int i)
 2{
 3    if(i<0)
 4        return NULL;
 5    LNode *s=L;
 6    int j=0;
 7    while(s!=NULL && j<i)
 8    {
 9        s=s->next;
10        j++;
11    }
12    return s;
13}

通过按位查找函数，指定结点后插入函数可以对按位增加进行修改。按位查找函数找到你所需要插入位序的前一个地址，在找到的地址进行下一个结点增加。

1bool InsertList(LinkList &L,int i,int e)
2{
3    if(i<0) //判断插入位置是否合法
4        return false;
5    LNode *p=GetElem(L,i-1);
6    return InsertNextNode(p,e);
7}

按数据查找结点

通过传入的参数，与链表的数据进行遍历对比，如果参数和遍历的数据相等，则返回该结点；如果没有一样的数据，最终会指向NULL，并返回。

1LNode * LocateElem(LinkList L,int i)
2{
3    LNode *s=L;
4    s=s->next;
5    while(s!=NULL && s->data != i)
6        s=s->next;
7    return s;
8}

链表长度

 1//求链表长度
 2int Length(LinkList L)
 3{
 4    int len=0;
 5    LNode *s=L;
 6    while(s->next!=NULL)
 7    {
 8        s=s->next;
 9        len++;
10    }
11    return len;
12}

单链表的建立

尾插法

​ 先建立一个单链表结构体L，在尾插法函数内引用L，给L分配一个结点的空间作为头节点。创建插入结点s，指向尾结点的结点p。变量x为特定值，则不再插入数据，否则x为插入值。由于最初单链表只有头节点，此时尾结点p指向头节点L。若x为插入值，则将x的值赋值给插入结点s，并使结点p的下一个结点为结点s，再令结点p指向新的尾结点s。以下为尾插法的代码。

 1LinkList InsertTail(LinkList &L)
 2{
 3    int x;	//插入值x
 4    L =(LinkList) malloc(sizeof (LNode));//头节点
 5    LNode *s;	//插入结点
 6    LNode *p=L;	//尾节点
 7    cin >> x ;
 8    while(x!=9999)
 9    {
10        s = (LNode *) malloc(sizeof (LNode));
11        s->data=x;
12        p->next = s;	//尾节点指向新的尾节点
13        p=s;	//p成为新的尾节点
14        cin >> x;
15    }
16    p->next=NULL;
17    return L;
18}
头插法

传入链表结构体L，给L分配空间，使其成为头节点，并使L指向的下一个结点为空。变量x为特定值则不执行插入操作，否则插入x值。执行插入结点的操作，给插入结点s分配空间，将x值赋值给结点s，并使结点s指向的下一个结点为头节点L所指向的下一个结点，再使头节点L指向的下一个结点为结点s。

 1LinkList InsertHead(LinkList &L)
 2{
 3    int x;	//变量x
 4    L =(LinkList) malloc(sizeof (LNode));//头节点
 5 	LNode *s;	//插入结点s
 6    L->next = NULL;
 7    cin >> x;
 8    while(x != 999)
 9    {
10        s =(LNode *) malloc(sizeof (LNode));
11        s->data = x;
12        s->next = L->next;
13        L->next = s;
14        cin >> x;
15    }
16    return L;
17}
