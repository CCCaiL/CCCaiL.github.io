单链表的相关要素

单链表

单链表结构体

1typedef struct LNode
2{
3    int *data;
4    struct LNode *next;
5}LNode,*LinkList;
  • LNode作为结构体LNode的别名，*LinkList作为结构体指针
  • LNode在使用上通常代指结点，*LinkList使用上作为链表

不带头结点单链表的初始化

1bool InitList(LinkList &L)
2{
3    L=NULL;
4    return true;
5}

带头结点单链表的初始化

1bool InitList(LinkList &L)
2{
3    L=(LNode *) malloc(sizeof (LNode));
4    if(L==NULL)
5        return false;
6    L->next=NULL;
7    return true;
8}

带头结点与不带头结点的区别

在使用上，不带头结点更为麻烦。不带头结点的头指针指向用于存放数据的结点，带头结点的头指针则指向头结点，该头结点不存放数据，头结点指向的下一个结点存放数据。
