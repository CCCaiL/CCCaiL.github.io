关于栈的学习

栈

栈是只允许在一端进行插入或删除操作的特殊线性表。

  • 栈顶是允许进行插入或删除操作的端
  • 栈底是不允许进行插入或删除操作的端
  • 元素遵守先进后出的准则。
结构体定义

定义一个数组用于存储数据，top则作为指向栈顶的下标。

1typedef struct
2{
3    int data[MaxSize];
4    int top;
5}Stack;
栈的初始化

由于最开始没有任何数据，此时top指向-1，表示空栈。

1void InitStack(Stack &S)
2{
3    S.top=-1;
4
5}
栈的判空

如果下标为-1，表示栈还没有存入数据为空。

1bool EmptyStack(Stack &S)
2{
3    if(S.top == -1)
4        return true;
5    return false;
6}
入栈（插入数据）
1bool InsertStack(Stack &S,int x)
2{
3    if(S.top == MaxSize-1)  //判断是否可以插入数据
4        return false;
5    S.top = S.top + 1;      //top+1，成为栈顶，插入数据
6    S.data[S.top] = x;
7    return true;
8}
出栈（删除数据）
1bool DeleteStack(Stack &S,int &x)
2{
3    if(S.top==-1)   //判断是否为空栈
4        return false;
5    x = S.data[S.top];
6    S.top--;
7    return true;
8}

读栈（读取栈顶元素）

1bool ReadStack(Stack &S,int &x)
2{
3    if(S.top==-1)   //判断是否为空栈
4        return false;
5    x = S.data[S.top];
6    return true;
7}
