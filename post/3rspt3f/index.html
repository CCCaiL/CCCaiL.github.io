C++类与对象的学习

结构体

在C++中，结构体可以定义变量与函数。

 2{
 3    string name;
 4    void print()
 5    {
 6        cout<<"学生姓名为:"<<name<<endl;
 7    }
 8};
 9
10int main()
11{
12    student test;
13    test.name="LiHua";
14    test.print();

16}

类

将事物的封装起来，称为类。

1class student	//定义一个简单的学生类
2{
3    public:     
4        string name;
5        void print()
6        {
7            cout<<"学生姓名为:"<<name<<endl;
8        }
9};
类的访问权限

public：公有的，类内类外都可以访问

protected：保护的，类外不可以访问，类内可以访问，子类可以访问

private：私有的，类外不可访问，类内可以访问，子类不可以访问

 1class student
 2{
 3    public:
 4        string name;
 5        void show()
 6        {
 7            cout<<name<<endl;
 8            cout<<money<<endl;
 9            cout<<tall<<endl;	//类内访问protected和private权限变量
10        }
11    protected:
12        int money;
13    private:
14        int tall=175;
15};
类的实例
 1class student
 2{
 3    private:

 5        int age;

 7        void student_Init(int set_age,char *set_name)

构造函数与析构函数

构造函数在创建对象时，会自动调用，对对象进行初始化工作。

  • 函数名与类名相同
  • 没有返回值，不能写void
  • 可以有参数，也可以函数重载

析构函数在销毁对象时，会自动调用，对对象进行清理工作。

  • 函数名为 ~加上类名

  • 没有返回值，不能有参数

  • 不能发生函数重载

     1class student
 2{
 3public:
 4    student(int set_age,string set_name): age(set_age), name(set_name)	//给age，name赋值
 5    {
 6        cout<<"调用构造函数"<<endl;
 7        cout<<name<<endl;
 8    }
 9
10    student(){};
11    ~student()
12    {
13        cout<<"调用析构函数"<<endl;
14    }
15    void show()
16    {
17        cout<<name<<endl;
18        cout<<age<<endl;
19    }
20private:
21    int age;
22    string name;
23};
24
25void test()
26{
27    student cai;
28}
29
30int main()
31{
32   test();	//跳出test函数作用域会自动执行析构函数。
33    return 0;
34}
拷贝构造函数与匿名函数
1void test()
2{
3    student cai(10,"cai");
4    student Long(cai);		//拷贝构造函数
5    student(11,"Lihua");	//匿名函数
6    Long.show();//输出10，cai
7}
拷贝构造函数

分为浅拷贝构造函数与深构造拷贝函数

  • 浅拷贝构造函数新旧对象还是共享同一片内存。

  • 深拷贝构造函数则是创建一个一模一样的新对象，不在同一片内存。

静态变量

静态成员变量在类内只能声明，不能定义；要在全局定义，在类内的声明静态成员变量是限制作用域。

1class Student
2{
3public:
4    static int a;	//静态成员变量声明
5};
6int Student::a=10;	//静态成员定义
单例模式

步骤:

  • 无参构造与拷贝构造的私有化。
  • 定义一个静态的类成员指针变量，并指向new出的一个对象。
  • 静态的类成员指针变量私有化，设置接口，调用该静态的类成员指针变量。
 1class Student
 2{
 3public:
 4    int age;
 5    static Student * getSingle()	//设置接口
 6    {
 7        return single;
 8    };
 9private:
10     static Student *single;	//静态的类成员指针变量
11     Student(){};	//无参构造与拷贝构造的私有化
12     Student(const Student &p){};
13};
14Student * Student::single = new Student;//new出一个对象
15int main()
16{
17    Student *cai = Student::getSingle();
18    cai->age=10;
19    Student *A = Student::getSingle();
20    cout << cai->age <<endl;	//输出10
21    cout << A->age <<endl;	//输出10
22    return 0;
23}

友元

友元函数是一种可以访问私有成员的特权函数。在类中，声明友元的全局函数为友元函数。友元函数即可调用私有变量

 1class Student
 2{
 3    friend void print_age(Student &C);
 4public:
 5    Student(int a):age(a)
 6    {
 7
 8    }
 9private:
10    int age;
11};
12
13void print_age(Student &C)
14{
15    cout<<C.age<<endl;
16}
17int main()
18{
19    Student C(10);
20    print_age(C);
21    return 0;
22}
