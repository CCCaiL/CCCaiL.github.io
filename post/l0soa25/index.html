C++的学习

C++的学习

冒号作用域

C++里冒号代表着作用域。下列代码输出结果为:

输出全局变量a:10

输出全局变量a:100

1int a = 10;
2
3int main()
4{
5    int a = 100;
6    cout << "输出全局变量a:"<<::a<<endl;
7    cout << "输出局部变量a:"<<a<<endl;
8    return 0;
9}

命名空间

通过命名空间的使用，可以避免函数名，变量名等等的冲突定义。控制函数等的作用域。命名空间支持嵌套定义。

命名空间的定义

namespace 空间名{};

1namespace Demo {
2    string A="定义一个命名空间";
3}
4
5namespace A {
6    namespace B {
7        string A="命名空间的嵌套定义";
8    }
9}
命名空间的调用

命名空间的调用方法有冒号作用域的调用与using关键字。

1namespace A {
2    int a= 10;
3}
4
5int main()
6{
7    cout<<A::a<<endl;   //冒号作用域调用命名空间
8    return 0;
9}

using关键字分为调用某个命名空间的某个标识符与直接调用该命名空间。

 1namespace A {
 2    int a = 10;
 3    int b = 100;
 4}
 5
 6int main()
 7{
 8    using A::a;      //using调用命名空间下的变量a
 9    cout<<a<<endl;
10    cout<<b<<endl;   //并没有调用变量b，无法读取
11    return 0;
12}

 1namespace A {
 2    int a = 10;
 3    int b = 100;
 4}
 5
 6int main()
 7{
 8    using namespace A;      //using调用命名空间
 9    cout<<a<<endl;
10    cout<<b<<endl;
11    return 0;
12}

当用using直接调用某个标识符时，该作用域存在相同标识符则会发生冲突；当using调用namespace时，存在相同标识符，则采用就近原则。

 1namespace A {
 2    int a = 10;
 3}
 4
 5int main()
 6{
 7    int a=1;
 8    using A::a;     //产生错误
 9    cout<<a<<endl;
10    return 0;
11}

 1namespace A {
 2    int a = 10;
 3}
 4
 5int main()
 6{
 7    using namespace A;
 8    int a=1;
 9    cout<<a<<endl;    //输出1
10    return 0;
11}

结构体的变化

在C++中，当你定义了一个结构体，定义结构体变量时，可以不使用struct关键字。

 1struct test
 2{
 3    int a;
 4};
 5
 6int main()
 7{
 8    test A;
 9    A.a=10;
10    cout<<A.a<<endl;
11    return 0;
12}

三目运算符

在c语言中，三目运算符返回的是一个常量；c++语言中，三目运算符返回的是一个变量。

1int main()
2{
3    int a=10;
4    int b=20;
5    (a>b ? a:b)=100;
6    cout<<b<<endl;  //输出结果为100
7    return 0;
8}

const关键字

在c语言当中，const定义局部变量，可以通过指针修改const变量的值。注意，当const定义的变量为全局变量时，保存在常量区，则不能通过指针修改。

1int main()
2{
3    const int a = 10;
4    int *p = &a;
5    *p=100;
6    printf("a=%d",a);   //输出为100；
7    return 0;
8}

1const int a = 10;	//const定义的全局变量
2int main()
3{
4    int *p = &a;
5    *p=100;
6    printf("a=%d",a);   //没有输出
7    return 0;
8}

C++中，const定义的局部变量则无法通过指针进行修改。

1int main()
2{
3    const int a=10;
4    int *p=(int *)&a;
5    *p=100;
6    cout<< a <<endl;    //a依旧为10
7    return 0;
8}

C++的引用

C++语言中，&在=的左边表示引用，右边表示取地址。本质上是取别名，指向某个标识。使用上，引用一旦初始化，就不能改变，不能为空引用。

 1void test(int &L)
 2{
 3    L=100;
 4}
 5
 6int main()
 7{
 8    int a=10;
 9    test(a);
10    cout<< a <<endl;    //a依旧为10
11    return 0;
12}
